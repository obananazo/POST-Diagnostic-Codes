# Common POST Diagnostic Codes by Manufacturer

## ASUS (LED/Keyboard Blink Patterns)
| Pattern | Issue |
|---------|-------|
| 1 blink | CPU not detected or failed |
| 2 blinks | Memory (RAM) not detected |
| 3 blinks | Graphics card/chip error |
| 4 blinks | Boot device not detected |
| 8-second flash cycle | RAM improperly seated or failed |
| Continuous rapid blinking | Power supply issue |

## Dell (Beep Codes)
| Pattern | Issue |
|---------|-------|
| 1 beep | BIOS ROM failure |
| 2 beeps | RAM not detected |
| 3 beeps | Motherboard failure |
| 4 beeps | RAM failure |
| 5 beeps | CMOS battery failure |
| 6 beeps | Video card failure |
| 7 beeps | CPU failure |

## HP (LED Blink Patterns)
| Pattern | Issue |
|---------|-------|
| 2 red blinks | Motherboard failure |
| 3 red blinks | CPU failure |
| 4 red blinks | RAM failure |
| 5 red blinks | Graphics card failure |
| 6 red blinks | Boot device error |
| 7 red blinks | Thermal issue detected |

## Lenovo (Beep Codes)
| Pattern | Issue |
|---------|-------|
| 1-3-3-1 | Memory test failure |
| 2-4-3-1 | Video card failure |
| 1-3-4-2 | Keyboard controller failure |
| 1-1-2 | CPU failure |
| 1-1-3 | CMOS read/write error |
| 2 short beeps | POST success (normal) |

## AMI BIOS (Traditional Beep Codes)
| Pattern | Issue |
|---------|-------|
| 1 short beep | Memory refresh failure |
| 2 short beeps | Parity error |
| 3 short beeps | Base 64K memory failure |
| 4 short beeps | System timer failure |
| 5 short beeps | CPU failure |
| 6 short beeps | Keyboard controller failure |
| 7 short beeps | Virtual mode processor exception |
| 8 short beeps | Display memory failure |
| 9 short beeps | ROM checksum error |
| 10 short beeps | CMOS shutdown register failure |
| 11 short beeps | Cache memory error |
| 1 long, 3 short | Memory test failure |
| 1 long, 8 short | Display test failure |
